# Gestión Villa il Fanale

Aplicación local y sin costo mensual para administrar consultas, reservas, preparación, ingresos, inventario y contenido de Villa il Fanale.

## Cómo abrirla

Abrí `index.html` con Chrome, Safari o cualquier navegador moderno. Los datos se guardan solamente en ese navegador y dispositivo.

La sección **Conexiones** permite guardar los enlaces de Facebook, Instagram y Airbnb, abrir Meta Business Suite, importar calendarios y configurar los datos de los comprobantes.

## Instalación y publicación gratuita

La aplicación ya incluye manifiesto y funcionamiento sin conexión. Para que Safari o Chrome ofrezcan **Instalar**, debe estar publicada en una dirección HTTPS. Puede alojarse como sitio estático en GitHub Pages, Cloudflare Pages o Netlify sin contratar un plan pago.

No se incluyeron DNI ni teléfono del propietario en los archivos públicos. Esos datos se cargan desde **Conexiones** y permanecen en el navegador.

## Importar calendarios

Desde el botón de flecha de la cabecera se puede seleccionar un archivo `.ics` exportado desde Airbnb o Calendario de Apple. Los eventos se incorporan como reservas externas.

La lectura directa de un enlace privado `.ics` puede ser rechazada por Airbnb debido a restricciones del navegador. La importación mediante archivo siempre queda disponible como alternativa gratuita.

## Privacidad

La aplicación no tiene servidor ni usuario/contraseña. No publiques esta carpeta en internet con datos reales de huéspedes: DNI, teléfonos y domicilios quedan guardados localmente en el navegador.
